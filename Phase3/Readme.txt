Run Backend first then frontend
Backend:
1.Install python along with pip
2.Go to backend folder (Phase3 -> CSE587P3 -> Phase3_app -> Backend)and run the following commands in commandline:
 pip install -r requirements.in
 pip install -r requirements.txt
 uvicorn api_DPDB:app --reload

Frontend:

1. Install npm
2. Go to form-app(Phase3 ->CSE587P3 -> Phase3_app -> form-app) folder
2. run following commands in commandline:
 npm install --legacy-peer-deps
 npm run start
